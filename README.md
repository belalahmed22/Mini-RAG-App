# 🚀 Mini-RAG-App

A lightweight **Retrieval-Augmented Generation (RAG)** application built with FastAPI.

## ✨ Features

- 📄 Document upload and processing
- 🔍 RAG query handling
- 🚀 FastAPI backend with automatic API docs
- 🐍 Python 3.8+

## 📋 Requirements

- Python 3.8 or later

## 🔧 Installation

### Option 1: Using MiniConda

1. Download and install MiniConda from [here](https://docs.anaconda.com/free/miniconda/#install)

2. Create a new environment:
   ```bash
   conda create -n mini-rag python=3.10 -yPython 3.8 Or Later

   conda activate mini-rag
